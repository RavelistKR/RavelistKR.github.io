# RavelistKR.github.io

> 의정부·양주 생활권을 중심으로 한 공인중개사 안내용 GitHub Pages 프로젝트  
> 현재는 **개업 준비 단계**를 전제로 구성한 소개형 부동산 홈페이지입니다.

---

## 프로젝트 소개

이 저장소는 공인중개사 홈페이지를 GitHub Pages로 운영하기 위한 정적 웹사이트 프로젝트입니다.

현재 홈페이지는 아래 흐름으로 구성되어 있습니다.

- 소개
- 지역분석
- 추천매물
- 실거래가·시세
- 부동산 정보
- 상담문의
- 오시는 길

핵심 방향은 **과한 영업형 표현보다**,  
의정부·양주 생활권을 **실거주 관점으로 차분하게 설명하는 홈페이지**를 만드는 것입니다.

---

## 기술 구성

- HTML
- CSS
- JavaScript
- GitHub Pages

빌드 도구 없이 바로 수정·배포 가능한 정적 사이트 구조입니다.

---

## 파일 구조

```bash
RavelistKR.github.io/
├── README.md
├── index.html
├── css/
│   ├── style.css
│   ├── components.css
│   ├── sections.css
│   └── responsive.css
└── js/
    └── main.js
```

### 파일 설명

- `index.html`  
  메인 랜딩 페이지입니다. 전체 사이트가 단일 페이지 구조로 작성되어 있습니다.

- `css/style.css`  
  전역 색상, 글꼴, 간격 등 기본 스타일을 관리합니다.

- `css/components.css`  
  헤더, 버튼, 카드, 내비게이션 같은 공통 UI 컴포넌트를 관리합니다.

- `css/sections.css`  
  hero, 지역분석, 추천매물, 실거래가·시세, 상담문의 등 섹션별 스타일을 관리합니다.

- `css/responsive.css`  
  모바일·태블릿 환경 대응용 반응형 스타일을 관리합니다.

- `js/main.js`  
  다크모드 토글, sticky 헤더, active 메뉴, 모바일 메뉴 스크롤 동작을 처리합니다.

## 주요 섹션

| 섹션 | ID | 설명 |
|---|---|---|
| 소개 | `#about` | 브랜드 소개와 첫 인상 |
| 지역분석 | `#region-analysis` | 의정부·양주 생활권 해석 |
| 추천매물 | `#listings` | 향후 실제 추천 매물 영역 |
| 실거래가·시세 | `#price` | 거래 흐름 및 시세 정보 영역 |
| 부동산 정보 | `#info` | 운영 상태 및 필수 정보 정리 |
| 상담문의 | `#contact` | 이메일, 문의 폼 연결 |
| 오시는 길 | `#map` | 향후 주소 및 지도 연결 |

## 수정 가이드

### HTML 수정
- 메뉴 구조 변경
- 섹션 문구 수정
- 상담/지도/정보 영역 텍스트 수정

→ `index.html`

### CSS 수정
- 전체 톤앤매너 수정 → `css/style.css`
- 헤더, 버튼, 카드 등 공통 UI 수정 → `css/components.css`
- 섹션별 디자인 수정 → `css/sections.css`
- 모바일 대응 수정 → `css/responsive.css`

### JavaScript 수정
- 다크모드
- sticky 헤더
- 현재 메뉴 active 처리
- 모바일 가로 스크롤 메뉴 보조 동작

→ `js/main.js`

## 배포

이 프로젝트는 GitHub Pages 기반 정적 사이트입니다.

기본적으로 `main` 브랜치의 정적 파일을 업데이트하면 페이지에 반영되는 구조로 운영할 수 있습니다.

## 메모

현재 사이트는 “준비 중인 공인중개사 홈페이지” 성격이 강하므로,  
초기에는 **소개 / 지역분석 / 상담 연결** 중심으로 운영하고,  
개업 후에 **추천매물 / 실거래가·시세 / 실제 사무소 정보**를 단계적으로 반영하는 흐름이 적합합니다.
