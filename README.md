# RavelistKR.github.io

의정부·양주 생활권을 중심으로 한 공인중개사 안내용 GitHub Pages 프로젝트입니다.  
현재는 **개업 준비 단계**를 전제로, 소개·지역분석·추천매물·실거래가·시세·부동산 정보·상담문의·오시는 길 구조로 홈페이지를 정리한 상태입니다.

## 개요

- 정식 개업 전: 소개, 지역분석, 상담 연결 중심으로 운영
- 개업 후: 추천매물, 실거래가·시세, 실제 운영 정보 반영 예정

## 파일 구조

- `index.html` — 메인 랜딩 페이지, 단일 페이지 구조
- `css/style.css` — 전역 색상, 타이포그래피, 간격
- `css/components.css` — 헤더, 버튼, 카드, 내비게이션 공통 스타일
- `css/sections.css` — hero, 지역분석, 추천매물, 시세, 상담문의 등 섹션별 스타일
- `js/main.js` — 다크모드, sticky 헤더, active 메뉴, 모바일 메뉴 스크롤 처리

## 주요 섹션

- `#about` — 소개
- `#region-analysis` — 지역분석
- `#listings` — 추천매물
- `#price` — 실거래가·시세
- `#info` — 부동산 정보
- `#contact` — 상담문의
- `#map` — 오시는 길
