# RavelistKR.github.io

의정부·양주 생활권을 중심으로 한 공인중개사 안내용 GitHub Pages 프로젝트입니다. 현재는 **개업 준비 단계**를 전제로, 소개·지역분석·추천매물·실거래가·시세·부동산 정보·상담문의·오시는 길 구조로 홈페이지를 정리한 상태입니다. [cite:1]

## 프로젝트 개요

이 저장소는 정식 개업 전 단계에서 사용할 소개형 부동산 홈페이지를 관리하기 위한 GitHub Pages 저장소입니다. 현재 `index.html`에는 상단 메뉴가 `소개`, `지역분석`, `추천매물`, `실거래가·시세`, `부동산 정보`, `상담문의`, `오시는 길` 순으로 구성되어 있고, 각 메뉴는 동일한 해시 섹션 구조로 연결됩니다. [cite:1]

## 현재 구조

### 파일 구성

- `index.html` — 메인 랜딩 페이지, 섹션 기반 단일 페이지 구조. [cite:1]
- `css/style.css` — 색상, 타이포그래피, 간격 등 전역 스타일 토큰 정의. [cite:1]
- `css/components.css` — 헤더, 버튼, 카드, 내비게이션 공통 컴포넌트 스타일. [cite:1]
- `css/sections.css` — hero, 지역분석, 추천매물, 실거래가·시세, 상담문의 등 섹션별 레이아웃 스타일. [cite:1]
- `js/main.js` — 다크모드 토글, 스크롤 시 헤더 상태, 현재 섹션 active 메뉴 처리용 스크립트. [cite:1]

### 메인 섹션

| 섹션 | id | 역할 |
|---|---|---|
| 소개 | `#about` | 첫 인상, 브랜딩, 핵심 소개 [cite:1] |
| 지역분석 | `#region-analysis` | 생활권 중심 지역 해석 [cite:1] |
| 추천매물 | `#listings` | 향후 추천 매물 노출 영역 [cite:1] |
| 실거래가·시세 | `#price` | 시세/거래 데이터 안내 영역 [cite:1] |
| 부동산 정보 | `#info` | 운영 상태, 향후 표기 정보 정리 [cite:1] |
| 상담문의 | `#contact` | 이메일, 문의 폼, 상담 연결 [cite:1] |
| 오시는 길 | `#map` | 향후 실제 사무소 위치 및 지도 연결 [cite:1] |

## 작업 가이드

### HTML 수정

주요 구조 변경은 `index.html`에서 진행합니다. 상단 메뉴와 각 섹션은 해시 기반으로 연결되어 있으므로, 메뉴명을 바꾸면 해당 `id`도 함께 점검하는 것이 좋습니다. [cite:1]

### CSS 수정

- 공통 디자인 수정: `css/style.css`, `css/components.css` [cite:1]
- 섹션별 디자인 수정: `css/sections.css` [cite:1]
- 모바일 메뉴, active 상태, sticky 헤더 관련 수정: `css/components.css` [cite:1]

### JS 수정

`js/main.js`에서는 현재 다크모드 토글과 active 메뉴 표시, 모바일 가로 스크롤 메뉴 보조 동작을 관리합니다. 내비게이션이 바뀌면 메뉴 선택자와 섹션 id 연결 상태를 먼저 확인하는 것이 안전합니다. [cite:1]
